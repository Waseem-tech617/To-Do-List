                                User Search Project

About:
This is a small JavaScript project where users can search profiles using an input field. As the user types in the search box, matching user cards appear instantly.
The goal of this project was to practice DOM manipulation and work with arrays in JavaScript.

Features:
* Live user search
* Dynamic card rendering
* Simple and clean layout

Built With:
* HTML
* CSS
* JavaScript

What I Practiced
While building this project I practiced:
* DOM manipulation
* Working with arrays
* Creating UI cards dynamically
* Basic project structure
